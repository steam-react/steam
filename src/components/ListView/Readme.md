ListView contains items arranged vertically or horizontally.

### Vertical

```js
<ListView type="vertical">
    <Button caption="First" />
    <Button caption="Second" />
    <Button caption="Third" />
</ListView>
```

### Horizontal

```js
<ListView type="horizontal">
    <Button caption="First" />
    <Button caption="Second" />
    <Button caption="Third" />
</ListView>
```
